## 6. Сумма чисел

Считайте строку (используйте input()), содержащую 1 натуральное число n

Найдите сумму всех чисел от 0 до n, которые делятся на 5, но не делятся на 3

**Sample Input 1:**

```commandline
100
```

**Sample Output 1:**

```commandline
735
```

**Sample Input 2:**

```commandline
422
```

**Sample Output 2:**

```commandline
11760
```
