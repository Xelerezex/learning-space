Откройте файл, имя которого передаётся на вход.

Файл содержит произвольное количество целых чисел, каждое с новой строки.

Выведите на печать сумму всех целых чисел, записанных в файле.

**Sample Input 1:**

```commandline
1.txt
```


**Sample Output 1:**

```commandline
3
```


**Sample Input 2:**

```commandline
2.txt
```


**Sample Output 2:**

```commandline
10
```


**Sample Input 3:**

```commandline
3.txt
```


**Sample Output 3:**

```commandline
45
```
