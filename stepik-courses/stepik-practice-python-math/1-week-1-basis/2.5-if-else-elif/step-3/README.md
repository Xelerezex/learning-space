## Римские цифры

Считайте число (см таблицу) в арабской записи и выведите на печать, соответствующую ему римскую цифру:

-    1 - I
-    5 - V
-    10 - X
-    50 - L
-    100 - C
-    500 - D
-    1000 - M


**Sample Input 1:**

```commandline
1

```

**Sample Output 1:**

```commandline
I
```


**Sample Input 2:**

```commandline
5
```


**Sample Output 2:**

```commandline
V
```
