Считывайте целые числа из потока ввода по одному до тех пор, пока не встретите строку "The End".

Выведите на печать сумму считанных чисел.

**Sample Input 1:**

```commandline
3
5
The End
```


**Sample Output 1:**

```commandline
8
```


**Sample Input 2:**

```commandline
5
42
6
-4
The End
9999
```


**Sample Output 2:**

```commandline
49
```
