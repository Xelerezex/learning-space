На вход подаётся 1 строка, слова в которой разделены пробелами.

Напечатайте слова из этой строки в обратном порядке, используя в качестве разделителя "-$-"

**Sample Input:**

```commandline
one two three four five six
```


**Sample Output:**

```commandline
six-$-five-$-four-$-three-$-two-$-one

```
