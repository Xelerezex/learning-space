Добавьте в ваш класс Animal метод echo, который принимает строку и 3 раза выводит её на печать, каждый раз с новой строки:

1й раз все символы в верхнем регистре

2й раз все символы как они переданы в метод

3й раз все символы в нижнем регистре

**Примечание.** В этой задаче не нужно ничего считывать и ничего выводить на печать. Только реализовать класс и метод.

Sample Input:
```commandline
Пиастры
```

Sample Output:
```commandline
ПИАСТРЫ
Пиастры
пиастры
```
