## 11

Считайте число n

Создайте единичную матрицу размера n, сохраните результат в переменную Z.


***Примечание.*** В этой задаче не нужно ничего выводить на печать. Только создать матрицу Z.

**Sample Input:**

```commandline
3
```

**Sample Output:**

```commandline
[[1. 0. 0.]
 [0. 1. 0.]
 [0. 0. 1.]]
```
