## Задача 4.2

В трамвайном парке имеются N трамваев маршрута №1 и M трамваев маршрута №2. Какова вероятность того, что K-м по счету на линию выйдет трамвай маршрута №1?

На вход подаётся 3 числа: N, M, K

Выведите на печать вероятность в процентах, округлив до 1 знака после запятой.

**Sample Input 1:**

```commandline
15
10
2
```

**Sample Output 1:**

```commandline
59.9%
```

**Sample Input 2:**

```commandline
10
10
2
```

**Sample Output 2:**

```commandline
50.1%
```

**Sample Input 3:**

```commandline
15
10
3
```

**Sample Output 3:**

```commandline
60.1%
```

**Sample Input 4:**

```commandline
4
1
5
```

**Sample Output 4:**

```commandline
80.0%
```

**Sample Input 5:**

```commandline
4
0
4
```

**Sample Output 5:**

```commandline
100.0%
```
