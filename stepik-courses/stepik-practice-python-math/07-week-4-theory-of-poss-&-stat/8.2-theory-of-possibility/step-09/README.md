## Задача 4.1

В трамвайном парке имеются N трамваев маршрута №1 и M трамваев маршрута №2. Какова вероятность того, что вторым по счету на линию выйдет трамвай маршрута №1?

На вход подаётся 2 числа: N, M

Выведите на печать вероятность в процентах, округлив до 1 знака после запятой.

**Sample Input 1:**

```commandline
15
10
```

**Sample Output 1:**

```commandline
60.0%
```

**Sample Input 2:**

```commandline
10
10
```

**Sample Output 2:**

```commandline
50.1%
```

**Sample Input 3:**

```commandline
0
5
```

**Sample Output 3:**

```commandline
0.0%
```

**Sample Input 4:**

```commandline
1
2
```

**Sample Output 4:**

```commandline
33.3%
```

**Sample Input 5:**

```commandline
2
0
```

**Sample Output 5:**

```commandline
100.0%
```
