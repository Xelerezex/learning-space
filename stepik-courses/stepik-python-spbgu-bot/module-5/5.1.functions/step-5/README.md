Мы для тебя подготовили весь код, который отвечает за ввод и вывод, тебе нужно только написать саму функцию.

**Входные данные**

В первой строке вводится число ```n```, а на следующей строке n вещественных чисел.

**Выходные данные**

Мода ```n``` чисел

**Sample Input:**

```commandline
9
6 2 6 6 8 9 9 9 0
```

**Sample Output:**

```commandline
6.0
```
