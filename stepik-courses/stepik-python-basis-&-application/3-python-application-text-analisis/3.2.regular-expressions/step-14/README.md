Вам дана последовательность строк.

В каждой строке поменяйте местами две первых буквы в каждом слове, состоящем хотя бы из двух букв.

Буквой считается символ из группы **\w**.

**Sample Input:**

```commandline
this is a text
"this' !is. ?n1ce,
```

**Sample Output:**

```commandline
htis si a etxt
"htis' !si. ?1nce,
```
