Даны два целых числа. Выведите значение наибольшего из них. Если числа равны, выведите любое из них.

**Формат входных данных**

Вводятся два числа.

**Формат выходных данных**

Выведите ответ на задачу.

**Sample Input 1:**

```cpp
1
2

```

**Sample Output 1:**

```cpp
2
```


**Sample Input 2:**

```cpp
5
3
```


**Sample Output 2:**

```cpp
5
```
