
Напишите программу, принимающую на вход целое число, которая выводит True, если переданное значение попадает в интервал (−15, 12] ∪ (14, 17) ∪ [19, +∞) и False в противном случае (регистр символов имеет значение).

**Sample Input 1:**

```python
20
```


**Sample Output 1:**

```python
True
```

```python
-20
```

